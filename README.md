# Netflix Content Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes Netflix content to uncover insights related to movies, TV shows, ratings, genres, countries, and release trends.

## Tools Used

* Power BI
* Power Query
* Excel

## Key Insights

* Movies represent nearly 70% of Netflix content.
* United States has the highest number of titles.
* TV-MA is the most common content rating.
* Content growth accelerated significantly after 2015.

## Dashboard Features

* Movies vs TV Shows Analysis
* Content Rating Distribution
* Top Genres Analysis
* Top Countries by Content Count
* Release Year Trend Analysis
* Interactive Filters and Visualizations

Netflix Titles Dataset
